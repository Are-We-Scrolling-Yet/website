This site will cover the scrolling and touchpad experience on Linux systems.

What's working

  • DE Touchpad Gestures

    Touchpad gestures are supported in GNOME and KDE.

  • Kinetic scrolling

    Kinetic/inertial/momentum scrolling on touchpads.
    • GTK4: Yes
    • GTK3: Yes, but scrolling is too fast.
    • Qt5/6: Yes on touch screens (https://bugs.kde.org/show_bug.cgi?id=456383)
    • Electron: Yes, needs a special flag, and is too fast, like GTK3.

  • Interpolated scrolling

    Interpolated (smooth) scrolling on mouse wheels.
    • GTK4: No
    • GTK3: No
    • Qt5/6: Yes
    • Electron: Yes

  • High resolution scroll wheel

Tips

Special tips to improve your scrolling experience!

Firefox

For the smoothest scrolling experience on Firefox using a touchpad, follow these instructions:
  1. Make sure you are using Wayland and that it is enabled. Starting from Firefox version 121, Wayland is enabled by default. More info on the Arch Wiki.
  2. Type about:config in the address bar and accept the risks.
  3. Touchpad scroll speed
    1. In the search bar, type mousewheel.default.delta_multiplier_ and replace the _x, _y and _z value with 15 (or adjust this number based on your liking).
  4. Mouse scroll speed (if this is not set, the mouse scrolling will be very slow because of the previous setting)
    1. In the search bar, type mousewheel.min_line_scroll_amount and set it to 300
  5. Overscroll bounce (similar to MacOS's bounce effect when you reach the end of a page)
    1. In the search bar, type apz.overscroll.enabled and set it to true
